# ID-Management-System

The project gives information about person identification details. 

It includes :
- Aadhar Details
- Insurance Details
- Ration Card Details
- Voting Details
- Bank Account Details
- Property Owned
- Passport Details
- Driving Licence Details

## To Run
- Create a Database using table_creation.php
- Create Tables and insert values into tables using tables_creation.php
- Then run index.html in a localhost
